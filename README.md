# Todo Board

A Kanban-style task management board built with TypeScript and Tailwind CSS.

![HTML](https://img.shields.io/badge/HTML-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?style=for-the-badge&logo=typescript&logoColor=white)
![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-38B2AC?style=for-the-badge&logo=tailwind-css&logoColor=white)

## Live Demo
[View Demo](https://subhiksha1196.github.io/todo-board/)

## Features
- Add tasks dynamically
- Check off tasks with strikethrough effect
- Tasks move to bottom when checked, back to top when unchecked
- In Progress column support

## Tech Stack
- TypeScript
- Tailwind CSS v3.9

## Status
🚧 Work in Progress — delete task feature coming soon

## Setup
```bash
npm install
npx tsc --watch
npx tailwindcss -i ./src/style.css -o ./docs/style.css --watch
```
